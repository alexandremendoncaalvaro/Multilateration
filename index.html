RTLS - Indoor Position Algorithm

Simulação de estimativa de posição por multilateração

O algoritmo funciona com o mesmo princípio do GPS, porém utilizando beacons no teto de um armazém no lugar de satélites.
Utilize o ponteiro do mouse para simular o deslocamento de uma pessoa no plano.
Assim como o ponteiro do mouse representa a posição real de uma pessoa, o ponto branco representa a posição estimada.
